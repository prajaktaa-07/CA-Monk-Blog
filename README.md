# CA Monk Blog Application

A modern blog application built using **React + TypeScript** that allows users to create, view, and manage blogs with a clean two-panel layout.

---

##  Features

*  View list of blogs
*  View blog details on the right side
*  Display blog image in detail view
*  Create new blogs
*  Clean and user-friendly UI
*  Fast and responsive layout

---

##  Tech Stack

* **Frontend:** React, TypeScript
* **Styling:** CSS (custom)
* **State Management:** React Hooks (`useState`, `useEffect`)
* **API Handling:** Axios
* **Build Tool:** Vite

##  How It Works

* Blogs are displayed in a list on the **left panel**
* Clicking the **View** button loads the blog details on the **right panel**
* Blog details include **image, title, author, and content**
* New blogs can be added using the **Create Blog** form
